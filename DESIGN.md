# Life.exe Design System

## 디자인 철학

> **"배경은 비우고, 색은 유저가 채운다."**

앱 자체는 미니멀 화이트. 유일한 컬러는 유저가 매일 칠하는 감정 픽셀.
앱이 화려한 게 아니라, 유저의 인생이 화려해지는 구조.

---

## 컬러 시스템

### 베이스 (앱 배경/구조)

| 토큰 | 색상 | 용도 |
|------|------|------|
| `--bg-primary` | `#FAFAFA` | 메인 배경 |
| `--bg-secondary` | `#F5F5F5` | 카드/섹션 배경 |
| `--bg-tertiary` | `#EFEFEF` | 비활성 영역 |
| `--border` | `#E5E5E5` | 구분선, 카드 테두리 |
| `--text-primary` | `#1A1A1A` | 제목, 강조 텍스트 |
| `--text-secondary` | `#6B6B6B` | 본문, 설명 |
| `--text-tertiary` | `#A0A0A0` | 힌트, 비활성 |

### 감정 색상 (유저가 칠하는 색 = 앱의 유일한 컬러)

| 감정 | 색상 | Hex | 톤 |
|------|------|-----|-----|
| 기쁨 (Joy) | 따뜻한 금색 | `#F5C842` | 밝고 따뜻 |
| 평온 (Calm) | 하늘색 | `#7EB8DA` | 차분 |
| 에너지 (Energy) | 민트그린 | `#6BC5A0` | 활기 |
| 피곤 (Tired) | 라벤더 | `#A78BBA` | 부드러운 무게 |
| 슬픔 (Sad) | 딥블루 | `#5B7BA5` | 깊은 |
| 분노 (Angry) | 코랄 | `#E87461` | 뜨거운 |
| 게으름 (Lazy) | 웜그레이 | `#B8B0A8` | 무채색 |

색상 선정 기준:
- 비비드/네온 금지 (CLAUDE.md 룰)
- 다크 뮤트까진 아니지만, 화이트 배경 위에서 **부드럽게** 보이는 톤
- 7가지 색이 격자에 나란히 놓여도 조화로워야 함
- 파스텔도 아님 -- 충분히 구분 가능한 채도

### 기록 안 한 날

| 상태 | 색상 | Hex |
|------|------|-----|
| 과거 (기록 안 함) | 연한 회색 | `#E0E0E0` |
| 미래 | 거의 투명 | `#F5F5F5` |
| 오늘 | 테두리 강조 | `#1A1A1A` 1px border |

## 레이아웃

### 메인 화면 구조

```
┌─────────────────────────────┐
│  Life.exe              [설정]│  <- 상단 바 (미니멀, 배경과 동일)
│                              │
│         34.2%                │  <- 히어로 숫자 (크게, 중앙)
│    당신의 인생 진행률          │  <- 서브텍스트
│                              │
│  ┌─────────────────────┐    │
│  │ ░░░░░░░░░░░░░░░░░░ │    │  <- 픽셀 격자 (스크롤/줌 가능)
│  │ ░░░░████░░░░░░░░░░ │    │     색칠된 부분 = 감정 색상
│  │ ░░░░████████░░░░░░ │    │     빈 부분 = 연한 회색
│  │ ░░░░░░░░░░░░░░░░░░ │    │
│  │ ░░░░░░░░░░░░░░░░░░ │    │
│  └─────────────────────┘    │
│                              │
│  오늘의 감정은?               │  <- 감정 선택 (하단)
│  😊 😌 ⚡ 😴 😢 😤 😐       │     7개 원형 버튼 (이모지 대신 색 원)
│                              │
│  ┌─────────────────────┐    │
│  │ 당신의 나이(28세)에...  │    │  <- 위인 카드
│  │ 마크 저커버그는         │    │
│  │ 페이스북을 상장시켰다   │    │
│  └─────────────────────┘    │
│                              │
│  [배너 광고]                  │
└─────────────────────────────┘
```

### 핵심 UI 원칙

1. **여백이 많아야 함** -- 미니멀 = 공간이 숨 쉼
2. **격자가 주인공** -- 화면의 40%+ 차지
3. **감정 선택은 한 줄** -- 7개 색 원, 탭 1번
4. **위인 카드는 조용히** -- 작은 카드, 읽고 싶으면 읽고
5. **숫자는 크게** -- 34.2%가 가장 먼저 눈에 들어와야

## 컴포넌트

### 감정 선택 버튼
- 지름 40px 원형
- 채워진 원 (해당 감정 색상)
- 선택 시: 원 안에 체크 + 살짝 커짐 (scale 1.15)
- 미선택: opacity 0.6
- 7개 가로 정렬, 균등 간격

### 픽셀 격자
- 각 픽셀: 4~8px 정사각형 (줌 레벨에 따라)
- gap: 1px (격자 사이 미세한 간격)
- border-radius: 1px (완전 직각은 아님)
- 오늘 픽셀: 1px solid #1A1A1A 테두리 + 살짝 펄스 애니메이션
- 줌아웃 시 픽셀이 작아지면서 추상화처럼 보임

### 위인 카드
- 배경: `--bg-secondary`
- border-radius: 16px
- padding: 20px
- 그림자 없음 (미니멀)
- 좌측에 카테고리 색 바 (4px width, 카테고리별 색상)
  - 기술 = 파랑, 예술 = 보라, 스포츠 = 초록, 비즈니스 = 금, 과학 = 민트, 음악 = 코랄

### 진행률 바
- 높이: 6px
- border-radius: 3px (full round)
- 배경: `--bg-tertiary`
- 채워진 부분: gradient (왼쪽 감정색 혼합)
- 부드러운 트랜지션 (width transition 0.5s ease)

### 공유 카드 (이미지 생성용)
- 1080x1920px (인스타 스토리)
- 배경: 화이트
- 상단: "Life.exe" 로고 + 퍼센트
- 중앙: 픽셀 격자 축소판
- 하단: 이번 달 감정 요약 (색상 비율 바)
- 폰트: Plus Jakarta Sans + Pretendard

### 온보딩
- 풀스크린 화이트
- 중앙: "Life.exe" 로고 (Plus Jakarta Sans, Bold)
- 아래: DatePicker (네이티브 또는 커스텀 휠)
- "시작하기" 버튼: #1A1A1A 배경, 화이트 텍스트, full-width, 56px 높이, radius 12px
- 전체적으로 **Apple 스타일 온보딩** -- 깔끔, 한 화면에 하나만

## 모션/인터랙션

| 인터랙션 | 모션 |
|----------|------|
| 감정 선택 | 탭한 원 scale(1.15) + 0.2s ease-out |
| 픽셀 색칠 | 오늘 픽셀이 회색->감정색으로 부드럽게 채워짐 (0.5s) |
| 오늘 픽셀 | 살짝 펄스 (opacity 0.8-1.0 반복, 2s) |
| 줌인/줌아웃 | 핀치 + 부드러운 스케일 트랜지션 |
| 위인 카드 등장 | 감정 선택 후 slide-up + fade-in (0.3s) |
| 공유 카드 | 격자 -> 카드 변환 애니메이션 (0.5s) |
| 페이지 전환 | 없음 (단일 화면 앱) |
